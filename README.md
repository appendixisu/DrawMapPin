# DrawMapPin
To draw map pin background image use core graphics.

![gif](/mappin.gif)

# How to use

I extension **UIIamge** to added a new function at *MapPin.swift* file.

##### Method:
```swift
func addMapPin(UIColor) -> UIImage
```

##### Example:
```swift
UIImage(named: "your_image")?.addMapPin(UIColor())
```
